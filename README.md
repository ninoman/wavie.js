### `Usage`

How we say, it's very easy to use WavieJS in your project

Load wavier.js <br/>

**index.html**
```html
<script src="wavie.js"></script>
```

Run wavie function 
**app.js**
```javascript
wavie('yourCanvasTagId', 'JSON config path');
```

You even can do this
**app.js**
```javascript
wavie(yourCanvasDOM, configObject);
```

**wavie.json**
```javascript
{
    "ballSize": 1,
    "speed": 1,

    "phaseRefX": 0.12,
    "phaseRefY": 0.12,

    "radiusX": 15,
    "radiusY": 15,

    "densityX": 6,
    "densityY": 6,

    "backgroundFrom":"#1e174a",
    "backgroundTo":"#000000",
    "degree":62,
    "ballColor": "#837ea2"
}
```
